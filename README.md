# Inject Header Tag

A [Flarum](http://flarum.org) extension. A back to top button.

## Installation

Install with composer:

```sh
composer require briantran/flarum-inject-header:"*"
```

## Updating

```sh
composer update briantran/flarum-inject-header:"*"
php flarum migrate
php flarum cache:clear
```
